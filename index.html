Tarheel Gallery

Tarheel Gallery is a website that brings together the fundamentals of HTML5, PHP, Javascript/JQuery, MySQL, Ajax cals, and RESTful interfaces. This website was built for the final project of my Advanced Web Programming course. The basis of Tarheel Gallery was to create a website which we were preferred to not use any kinds of frameworks or plugins.

Fully Integrated User System

I've showcased my ability to work with PHP and MySQL through the creation of a site-wide user system. I integrated a full user system into the website. User registration data is saved to the data base and logins are tracked through sessions and cookies. Users are able to be recognized through a cookie-based authentication system where only the admins know the name and configuration of the cookie.

Full Ajax Implementation

Tarheel Gallery demonstrates the use of Ajax calls through JQuery. In many aspects of the website, from user registration/logins or to fetching data for displaying the gallery on the main index page or the individual galleries for each user, Ajax calls are used to retrieve the information regarding those objects from the server and then the controller interface, usually through JavaScript, which is used to layout these objects onto the Users view through the browser.

Ascensions Laboratory

Ascensions Laboratory is a game made to tackle the fears of the human mind and help a person with deep fears to overcome them. Built for the final project of my Game Design class, Ascensions Laboratory was able to give me an introduction into game development. It is built and compiled using the Unity Game Engine and scripted through JavaScript

Event-Driven Programming

Unity has the ability to be scripted using a number of given languages. We decided to use JavaScript and this project helped further my knowledge of the language and shows my skills at event driven programming. Different objects were created and triggers were attached to these objects through JavaScript, such as an invisible plane, once these objects were hit or passed, they would trigger functions in JavaScript to be executed which gave the game it's responsiveness to user interaction.

Physiscs-Engine System

Unity comes with its own physics engine and with the use of JavaScript we were able to utilize this engine to move objects with the use of vectors, algebra, and physics itself. Objects were given velocity vectors and weights which all put together created a world that wasn’t beyond that of reality. Ascensions Island further shows the development and understanding of Unity's Physics Engine coupled with JavaScript

Practically my whole life story...

Hello! My name is Danyal Fiza. The following story is my self-taught journey to becoming a student of computer programming.

As a young human, I was endlessly fascinated by computers and technology. My fascination can be traced back to 1999, in a little store called “Best Buy.” My family was in need of a computer featuring the capabilities necessary for my older brothers’ school projects. Strolling through all of the Windows 95 machines, I felt so small. These big, gray machines towered my 7 year-old frame. As my siblings finally chose their machine and reached checkout, I remember staring at the large box as if it was a Christmas present just waiting to be unwrapped of its potential. And ever since, I was hooked, or wired. Little did I know this Windows 95 machine would be the first computer I would learn to experiment and explore my passion. With 512MB of ram, 8GB worth of hard drive space, this computer roared with seemingly infinite capabilities; it dueled as an engine and my classroom. Fast-forward years of adolescence and computer advancements, I was spending more and more time on my computer soaking up all the new technology and its inner workings.

At 12, my brother facilitated my computer science experience with the introduction to Adobe Photoshop. Beginning with the basics, I photo-shopped cars and googled car tutorials, until the day I found myself rapt in the search for website templates tutorials. From there on, I learned HTML code. Realizing that was insufficient education, I was left unsatisfied with the resulting ugly website creations. With the web as my teacher, I was driven to learn more about computer programming. The more I googled, the more I learned and was able to teach myself how to program all the tests and projects from my child to my adult years.

After enrolling at UNC, I admittedly did not know what I wanted to do. Yet, I realized one consistent thing in my life: computers. My love for computers and interest in their inter-workings stayed constant though out my studies. During my freshman year, I decided to take COMP 101: Intro to Programming, and I loved it! It was the best class I’d taken because it fulfilled my curiosity and passions for technology and offered a valuable skill set for my future. As I explored other areas of study (science and business), nothing stuck to me more than the COMP classes. At last, I decided to pursue my love of computer science. Now, I am excited to continue my journey and progress as a web designer, developer, and engineer!
